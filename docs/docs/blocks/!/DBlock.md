# DBlock

```clojure
(DBlock
  :Name [(String)]
  :Parameters [(Seq [(Any)])]
)
```

## Definition
A dynamic block.

## Parameters
| Name | Type | Default | Description |
|------|------|---------|-------------|
| Name | `[(String)]` | `""` | The name of the block to wrap. |
| Parameters | `[(Seq [(Any)])]` | `[]` | The parameters to pass to the wrapped block. |
