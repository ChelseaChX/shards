# Reflect.Blocks

```clojure
(Reflect.Blocks)
```


## Definition

Given a chain as input it will recurse deep inside it and gather all blocks in the chain, its sub-chains and sub-flows.


## Input

| Type | Description |
|------|-------------|
| `[(Chain)]` |  |


## Output

| Type | Description |
|------|-------------|
| `[(Seq [(Block)])]` |  |


## Examples

```clojure
(Reflect.Blocks

)
```
