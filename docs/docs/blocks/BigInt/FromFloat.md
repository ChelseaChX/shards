```clojure
(BigInt.FromFloat

)
```

## Parameters
| Name | Type | Default | Description |
|------|------|---------|-------------|
| ShiftedBy | `[(Int)]` | `0` | The shift is of the decimal point, i.e. of powers of ten, and is to the left if n is negative or to the right if n is positive. |
